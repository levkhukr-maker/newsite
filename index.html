Швидкі уроки

1. Змінні та Консоль

Створення змінних та вивід даних.

let name = "Jinzo";
console.log("Привіт, " + name);

2. Умови (If/Else)

Перевірка умов у коді.

let age = 18;
if (age >= 18) {
  console.log("Доступ дозволено");
} else {
  console.log("Замало років");
}

3. Цикли (Loops)

Повторення дій кілька разів.

for (let i = 1; i <= 5; i++) {
  console.log("Крок №" + i);
hash;}
